We'll build a swarm algorithm that traces shapes in an image by having a "bot" start at every pixel in the top row. Each bot moves downward (row by row) by choosing the neighbor in the next row whose color is most similar to its current color. The path each bot takes forms a continuous line that follows color‑consistent regions, collectively outlining the objects in the image.

Algorithm Overview
Initialization: For each column in the top row (row = 0), create a bot.

Movement: At each step, the bot looks at the three downward neighbors:

down‑left: (row+1, col-1)

down: (row+1, col)

down‑right: (row+1, col+1)
Only neighbors inside the image bounds are considered.

Color similarity: Compute the Euclidean distance in RGB space between the bot's current pixel and each candidate neighbor. The bot moves to the neighbor with the smallest distance (most similar color).

Path recording: Append each visited pixel to the bot's path.

Termination: The bot stops when it reaches the bottom row (row = height‑1).

After all bots have finished, we have a set of paths – one for every starting column – that together reveal the vertical “flow” of similar colors. These paths can be visualized directly, or further processed to extract object boundaries.

import cv2
import numpy as np
import matplotlib.pyplot as plt

def trace_paths(image):
    """
    Trace paths from every pixel in the top row downward.
    Returns a list of paths, each path is a list of (row, col) tuples.
    """
    # Convert to RGB if needed (OpenCV loads BGR by default)
    if len(image.shape) == 3 and image.shape[2] == 3:
        img_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
    else:
        img_rgb = image  # assume already RGB or grayscale

    h, w = img_rgb.shape[:2]
    paths = []

    # For each starting column
    for start_col in range(w):
        path = [(0, start_col)]
        r, c = 0, start_col

        while r < h - 1:
            # Candidate positions in the next row
            candidates = []
            for dc in [-1, 0, 1]:
                nc = c + dc
                if 0 <= nc < w:
                    candidates.append((r+1, nc))

            if not candidates:
                break  # no valid downward move (should not happen at top)

            # Current pixel color
            current_color = img_rgb[r, c]

            # Compute color distances (Euclidean in RGB)
            best_dist = float('inf')
            best_next = None
            for nr, nc in candidates:
                next_color = img_rgb[nr, nc]
                dist = np.linalg.norm(current_color - next_color)
                if dist < best_dist:
                    best_dist = dist
                    best_next = (nr, nc)

            # Move to the chosen neighbor
            r, c = best_next
            path.append((r, c))

        paths.append(path)

    return paths

def visualize_paths(image, paths, output_path='swarm_paths.png'):
    """Draw all paths on a copy of the image and save/display."""
    # Use a copy to avoid modifying original
    vis = image.copy()
    # If image is BGR (OpenCV default), convert to RGB for matplotlib
    if len(vis.shape) == 3 and vis.shape[2] == 3:
        vis_rgb = cv2.cvtColor(vis, cv2.COLOR_BGR2RGB)
    else:
        vis_rgb = vis

    # Draw each path as a thin line
    for path in paths:
        points = np.array(path, dtype=np.int32)
        for i in range(1, len(points)):
            cv2.line(vis_rgb, tuple(points[i-1][::-1]), tuple(points[i][::-1]), (255,0,0), 1)

    plt.imshow(vis_rgb)
    plt.axis('off')
    plt.savefig(output_path, bbox_inches='tight', pad_inches=0)
    plt.show()
    print(f"Visualization saved as {output_path}")
